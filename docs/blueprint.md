# **App Name**: NutriScan Pro

## Core Features:

- Real-time Barcode/QR Code Scanning: Utilize the device camera to scan barcodes and QR codes for real-time product detection. Integrates a browser-based scanning library for broad compatibility.
- Open Food Facts API Integration: Fetch product nutritional data from the Open Food Facts API using the scanned barcode, providing a comprehensive dataset of product information.
- Comprehensive Product Information Display: Display detailed product information including product name, brand, health score, nutritional breakdown, ingredients list, allergen highlighting, processing level, and nutritional quality score.
- Health Score Calculation: Calculate a health score (1-10) based on sugar content, sodium levels, fiber content, additives, processing level, and protein content, providing a quick health assessment.
- Dietary Restriction Analysis Tool: Analyzes product data to automatically identify allergens or non-compliant ingredients. Alerts user based on their selected dietary preferences (vegan, gluten-free, diabetic-friendly, low-sodium, keto). The tool warns users, and incorporates any info shown in the warnings, using generative AI when choosing the most relevant factors.
- Scan History Management: Maintain a scan history stored in local storage, allowing users to review previously scanned products and manage their scan history.
- User Preference Management: Allows the user to specify their own dietary preferences so relevant warnings can be issued if foods scanned do not match dietary guidelines. 

## Style Guidelines:

- Primary color: Fresh green (#77B35D) to evoke feelings of health and naturalness.
- Background color: Light pale green (#E6F2E0), provides a calming and neutral backdrop that ensures readability and allows the primary and accent colors to stand out without being overpowering.
- Accent color: Sunny yellow (#FFD700) highlights important information and actions with energy.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible readability.
- Use clear and informative icons to represent nutritional information and dietary restrictions.
- Prioritize a clean, intuitive layout that makes scanning and information retrieval seamless.
- Subtle animations enhance user engagement during scanning and product information display.
